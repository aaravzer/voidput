# voidput
## dependencies
### curl, perl.
## To install/setup(I think this works on most unix based machines.)
`git clone https://github.com/bigpecks/voidput.git && cd voidput && chmod +x voidpet.sh`

## To use:
`./voidpet.sh <voidpet username here>`
